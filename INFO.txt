  INFORMAÇÕES GERAIS
═══════════════════════════════════════════════════════════════════════════

📋 Nome do Projeto:
   Sistema de Registro de Distribuição Espacial de Animais

🎯 Objetivo:
   Aplicação web para coleta e análise de dados sobre distribuição de cães
   e gatos no município de Nova Iguaçu/RJ para pesquisas veterinárias e
   estudos epidemiológicos.

🏢 Desenvolvido para:
   Pesquisa veterinária - Nova Iguaçu/RJ

📄 Licença:
   MIT License

═══════════════════════════════════════════════════════════════════════════

  1. Execute o arquivo setup.bat

═══════════════════════════════════════════════════════════════════════════
  ESTRUTURA DE CAMPOS
═══════════════════════════════════════════════════════════════════════════

OBRIGATÓRIOS:
  • URB (Unidade de Registro Básico) - Ex: URB-001
  • Localidade/Bairro - Ex: Centro, Posse, Comércio
  • Endereço Completo - Ex: Rua das Flores, 123
  • Quantidade de Cães (Macho e Fêmea) - Número >= 0
  • Quantidade de Gatos (Macho e Fêmea) - Número >= 0
  • Data do Registro - Formato: DD/MM/AAAA
  • Nome do Tutor - Nome completo
  • Telefone - Formato: (XX) XXXXX-XXXX

OPCIONAIS:
  • Coordenadas GPS (Latitude e Longitude)

═══════════════════════════════════════════════════════════════════════════
  CARACTERÍSTICAS ESPECIAIS
═══════════════════════════════════════════════════════════════════════════

✅ FUNCIONA OFFLINE
   Todos os dados são armazenados localmente no navegador.
   Não requer conexão à internet, exceto para carregar mapas.

✅ PRIVACIDADE TOTAL
   Nenhum dado é enviado para servidores externos.
   Você tem controle total sobre seus dados.

✅ RESPONSIVO
   Interface otimizada para desktop, tablet e celular.
   Botões grandes para uso em campo.

✅ VALIDAÇÕES ROBUSTAS
   Sistema de validação completo previne erros de entrada.
   Feedback visual imediato para o usuário.

✅ MÚLTIPLAS EXPORTAÇÕES
   8 formatos diferentes de exportação disponíveis.
   Compatível com Excel, SPSS, QGIS, etc.

═══════════════════════════════════════════════════════════════════════════
  DICAS IMPORTANTES
═══════════════════════════════════════════════════════════════════════════

⚡ PERFORMANCE
   O sistema suporta até 10.000 registros sem problemas de performance.

💾 BACKUP
   Faça backup regular dos dados exportando em formato JSON.
   Vá em Análises → Backup Completo.

📱 USO MOBILE
   Para usar em campo, permita acesso à localização do navegador.
   Use "Capturar GPS" para adicionar coordenadas.

🔒 SEGURANÇA
   Não use modo anônimo/privado do navegador.
   Os dados serão perdidos ao fechar o navegador.

═══════════════════════════════════════════════════════════════════════════
  PROBLEMAS COMUNS E SOLUÇÕES
═══════════════════════════════════════════════════════════════════════════

❌ Erro ao instalar
   Solução: npm cache clean --force && npm install

❌ Porta 3000 já em uso
   Solução: O Vite escolherá outra porta automaticamente

❌ GPS não funciona
   Solução: Use HTTPS e permita acesso à localização

❌ Dados sumiram
   Solução: Não use modo anônimo. Faça backup regular.

❌ Mapa não carrega
   Solução: Verifique sua conexão com internet
